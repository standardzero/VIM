1. 插件管理利器 vim-plug

1.1 安装插件管理器

curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

1.2 配置插件管理器

安装插件，你需要在~/.vimrc文件中声明，具体格式如下：

call plug#begin('~/.vim/plugged')
" 需要安装的插件
call plug#end()

执行:source ~/.vimrc 使脚本生效。

1.3 安装插件

安装插件：vim-airline

call plug#begin('~/.vim/plugged')
" 需要安装的插件
Plug 'vim-airline/vim-airline'
call plug#end()

使用命令 :PlugStatus 查看状态

使用命令 :PlugInstall 安装插件

安装完成后，若需自定义插件，可以在配置文件中进行配置。

" ===== airline =====
set t_Co=256
set laststatus=2
let g:airline#extensions#tabline#enabled = 1
let g:airline#extensions#tabline#formatter = 'unique_tail'
let g:airline_powerline_fonts = 1

1.4 更新插件

:PlugUpdate：更新插件

:PlugDiff：查看更新状态

1.5 回滚插件

输入 :PlugDiff命令，然后按回车键查看上次 :PlugUpdate的更改，并在每个段落上按X将每
个插件回滚到更新前的前一个状态。

1.6 卸载插件

注释掉Plug 命令的插件和其对应的配置，输入命令:PlugClean。
